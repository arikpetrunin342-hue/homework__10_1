# homework__10_1

учебный проект по работе с GitHub и списками словарей в Python.

## Функционал

- **Фильтрация данных**: функция `filter_by_stake` отбирает записи по статусу (по умолчанию - `'EXECUTED'`).
- **Сортировка данных**: функция `sort_by_date` сортирует записи по дате. По умолчанию сортировка идёт по убыванию, но можно изменить порядок.

## Технологии

- Python 3.14
- Стандартная библиотека (`operator.itemgetter`)

## Установка и запуск

1. Клонируйте репозиторий. 
2. Создайте и активируйте виртуальное окружение:
    ```bash
    python -m venv .venv
    source .venv/bin/activate  # Для Windows: .venv\Scripts\activate
3. Запустите скрипт:
    `python src/processing.py`
